WMO Climate Monitoring Dashboards

The WMO generates a set of climate indicators based on global data sets. The links below show these indicators in a set of dashboards covering key indicators and regional temperature indicators.

The dashboard are created to support production of the WMO State of the Global Climate reports and the WMO Regional State of the Climate reports.
